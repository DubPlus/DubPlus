# DubPlus
Dub+ - A Dubtrack.fm script/extension for added features and customizations

-----------------
Features
---

Main
- AutoVote
- AFK Auto-respond
- Emotes
- AutoComplete Emoji
- Custom Mentions
- Notify on Mentions
- SpaceBar Mute
- Warn on navigation
- {Updubs, Grabs} in Chat

User Interface
- FullScreen
- Split Chat
- Hide {Chat, Video, Avatars, Background}
- Show TempStamps

Customize
- Community Theme
- Custom CSS
- Custom Background
- Snow Animation
