### Description

A Dubtrack.fm and QueUp.net script/extension for added features and customizations to improve your Dubtrack/QueUp experience

Features include:

- AutoVote
- AFK Auto-respond
- Emotes
- AutoComplete Emoji
- Custom Mentions
- Notify on Mentions
- Notify on PM - NEW!
- DJ Notification - NEW!
- SpaceBar Mute
- Warn on navigation
- Dubvote Info on Hover
- Dubvote Info in Chat
- FullScreen
- Split Chat
- Hide Elements
- Show Timestamps
- Community Theme
- Custom CSS
- Custom Background
- Custom Notification Sound
- Snow Animation
- Rain Animation

More info at https://dub.plus
